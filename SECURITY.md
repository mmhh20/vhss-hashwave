# Security policy

VHSS is research software, not production cryptocurrency, wallet, mining-pool,
or smart-contract infrastructure.

## Supported version

Only the latest tagged release is supported for security fixes.

## Reporting

Do not publish an exploit involving share verification, target comparison,
serialization ambiguity, or denial of service before contacting the maintainer.
Add a private security-reporting channel to the public repository before release.

## Non-goals

The project has not been audited for custody of funds, adversarial mining pools,
public consensus, token economics, or internet-facing deployment.
