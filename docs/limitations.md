# Limitations

- VHSS is not a large language model and does not understand unrestricted prompts.
- Hash rarity is statistically unrelated to candidate quality.
- Software experiments found no significant search advantage for hash seeds over
  a conventional pseudo-random source under the tested conditions.
- Current benchmarks are small proof-of-correctness tasks, not a comprehensive
  symbolic-regression leaderboard.
- The hyperdimensional classifier is a limited demonstration.
- No physical miner, Stratum deployment, token economy, public consensus system,
  or blockchain security analysis is included.
- Deterministic reproduction depends on the specified software version and task.
- Tests reduce known defects but cannot prove absence of all defects.
